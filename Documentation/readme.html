Getting Started Adding Zoomy to your site

Zoomy.js was built from the ground up with efficency and cross browser compatibilty in mind. Zoomy.js is perfect for image galleries that require that little bit extra to make the user experiance more pleasant and intuitive or just a simgle image on a page.

Markup

To get the ball rolling, you need very little markup.

Its so simple, you don't even need to specify a large image for it to work. It will detect your set up and use the thumbnail instead.

Recommended

<div class="some-class" data-zoomed-image="/path/to/zoomed-image.jpg">
  <img src="/path/to/thumb.jpg" />
</div>

Bare Minimum

<div class="some-class">
  <img src="/path/to/thumb.jpg">
</div>
Note! You should only use the bare minimum if you have a large image resized to a thumbnail size. Otherwise, it will look odd!

Javascript

Once you have the markup out for the way, setup is Easy!

Default

$('.some-class').zoomy();

With Options

$('.some-class').zoomy({
  option: 'property',
  option: 'property'
});
More on options

Multiple Instances on the Same Page

Its likely that you will want to have more than one instance of Zoomy.js on one page, however this can present a few issues when each instance requires different options as they will get overwritten.

Good

Zoomy.js supports HTML data attributes, which means you can define options on the element itself in the markup. Also, from a performance perspective you no longer have the overhead of multiple calls to the same plugin. More on Data attributes

// ­se a class as a hook for all your elements
$('.zoomed').zoomy({
  innerZoom: true, // You can still set global options here, then override existing or add more to the element
  lensHeight: 100,
  lensWidth: 120
});
 
// Using data attributes to store the options
<div class="zoomed" data-zoomed-image="/path/to/zoomed-image.jpg" data-inner-zoom="false">
  <img src="/path/to/thumbnail.jpg" />
</div>

Bad

Instaniating the plugin on multiple selectors is not recommended and will increase page load times!

// First Object with its own properties
$('#zoomed-1').zoomy({
  innerZoom: true,
  lensHeight: 100,
  lensWidth: 120
});
   
// Second Object with its own properties
$('#zoomed-2').zoomy({
  popover: {
    width: 400,
    height: 200
  }
});

Options Customize the look and functionality with options

Zoomy API

Zoomy is highly configurable, here is a list of the options you can customize:

Option Description HTML5 Data Values
popover className Customise the class name of the popover Not available Any valid selector
popoverHeight Height of the popover data-popover-height Any number value
popoverWidth Width of the popover data-popover-width Any number value
popoverPosition Position of the popover relative to the element data-popover-position 'top', 'left', 'bottom', 'right'
popoverOffsetX Horizontal offset of the popover data-offset-x Any number value
popoverOffsetY Vertical offset of the popover data-offset-y Any number value
lensHeight Height of the lens (used only when inner zoom enabled) data-lens-height Any number value
lensWidth Width of the lens (used only when inner zoom enabled) data-lens-width Any number value
magnify Level of magnification on large image data-magnify Any number value
(less than 1 to zoom out)
triggerEvent The event in to trigger the zoom state data-trigger-event 'hover', 'click'
innerZoom Show the zoomed image in the lens instead of popover data-inner-zoom true, false
preload Preloads the large images on page load data-preload true, false
duration Duration for the transition events in milliseconds data-duration Any number value.
Higher value = slower

Defaults

Note! On options where there is no value specified, the value is pre calculated 
defaults = {
    popover: {
        className: 'popover',
    },
    popoverHeight: , // Thumbnail height * 1.5
    popoverWidth: , // Thumbnail width * 1.5,
    popoverPosition: 'right',
    popoverOffsetX: 40,
    popoverOffsetY: 0,
    lensHeight: , // Thumbnail height / 1.5
    lensWidth: , // Thumbnail width / 1.5
    magnify: 1,
    triggerEvent: 'hover',
    innerZoom: false,
    preload: true,
    showPreloader: true,
    duration: 100,
}

Examples Configuration possibilities

Default Configuration

Zoomy.js will work straight out of the box. All you need is a div with an image in it. Simply instantiate the plugin on the containing div.

By default, the object will preload and use a popover to display the large image among an array of other defaults.

$('.zoom').zoomy();

Inner Zoom

Instead of having a popover, you can use the lens to accomodate the zoomed image so it acts like a magnifiing glass. By default, the size of the lens is 1.5x smaller than the thumbnail, but this can be overwritten to anything you want. The larger it is the less movable it is.

$('.zoom').zoomy({
  innerZoom: true,
  lensHeight: 60,
  lensWidth: 100
});

Magnify

Achieve an even greater level of zoom by using the magnifiy property by specifiy how much you want to increase the image size by

You can even zoom out the image by using numbers less than 1. Using 0.5 would reduce the image by 50%

This can work particularly well when not specifying a large image using 'data-zoomed-image'.

$('.zoom').zoomy({
  magnify: 2
});

No Large Image Specified - Use Self

Zoomy.js will detect if a large image is specified or not. If not then it will use the thumbnail. This can be handy if your thumbnail is just a large image resized with css.

Inner Zoom with custom Lens Size

Custom Popover size

Combination of Popover Offset and Size

By using these to properties you can place the popover anywhere on the page relative to the thumbnail

$('.zoom').zoomy({
  popoverOffsetX: 0,
  popoverOffsetY: 50,
  popoverPosition: top,
  popoverWidth: 940,
  popoverHeight: 630
});

Trigger Event - Click

Active the zoom by clicking on the image

$('.zoom').zoomy({
  triggerEvent: 'click'
});

Fade Duration

Set the fade duration to be as quick or as long as you want

$('.zoom').zoomy({
  duration: 500
});

Notes & Tips Just a few tips to help out

  • Make sure the img in thumbnail and popover DO NOT have max-width: 100% (Acts weird in older browsers)
  • Do not give the popover any padding
  • Don's use excessive border on the lens or padding or margin
  • Make sure you dont have any other elemens on the page with the same name as the lens, popover
  • If relying on the 'use self' option, you need to make sure you resize it with css or by adding the height and width attributes to the image itself
