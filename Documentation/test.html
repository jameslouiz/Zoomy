Zoomy in Action

Default Configuration

Zoomy.js will work straight out of the box. All you need is a div with an image in it. Simply instantiate the plugin on the containing div. By default, the object will preload and use a popover to display the large image among an array of other defaults.

$('.zoom').zoomy();

<div class="zoom" data-zoomed-image="path/to/large/image.jpg">
    <img src="path/to/small/image.jpg" alt="Thumbnail">
</div>

Inner Zoom

Instead of having a popover, you can use the lens to accomodate the zoomed image so it acts like a magnifiing glass. By default, the size of the lens is 1.5x smaller than the thumbnail, but this can be overwritten to anything you want. The larger it is the less movable it is.

$('.zoom').zoomy({
  innerZoom: true,
  lensHeight: 60,
  lensWidth: 100
});

Magnify

Achieve an even greater level of zoom by using the magnifiy property by specifiy how much you want to increase the image size by You can even zoom out the image by using numbers less than 1. Using 0.5 would reduce the image by 50% This can work particularly well when not specifying a large image using 'data-zoomed-image'.

$('.zoom').zoomy({
  magnify: 2
});

No Large Image Specified - Use Self

Zoomy.js will detect if a large image is specified or not. If not then it will use the thumbnail. This can be handy if your thumbnail is just a large image resized with css.

Inner Zoom with custom Lens Size

Custom Popover size

Combination of Popover Offset and Size

By using these to properties you can place the popover anywhere on the page relative to the thumbnail.

$('.zoom').zoomy({
  popoverOffsetX: 0,
  popoverOffsetY: 50,
  popoverPosition: top,
  popoverWidth: 940,
  popoverHeight: 630
});

Trigger Event - Click

Active the zoom by clicking on the image

$('.zoom').zoomy({
  triggerEvent: 'click'
});

Fade Duration

Set the fade duration to be as quick or as long as you want

$('.zoom').zoomy({
  duration: 500
});

Zoomy API

Zoomy is highly configurable, here is a list of the options you can customize:

Option Description HTML5 Data Values
popover className Customise the class name of the popover Not available Any valid selector
popoverHeight Height of the popover data-popover-height Any number value
popoverWidth Width of the popover data-popover-width Any number value
popoverPosition Position of the popover relative to the element data-popover-position 'top', 'left', 'bottom', 'right'
popoverOffsetX Horizontal offset of the popover data-offset-x Any number value
popoverOffsetY Vertical offset of the popover data-offset-y Any number value
lensHeight Height of the lens (used only when inner zoom enabled) data-lens-height Any number value
lensWidth Width of the lens (used only when inner zoom enabled) data-lens-width Any number value
magnify Level of magnification on large image data-magnify Any number value
(less than 1 to zoom out)
triggerEvent The event in to trigger the zoom state data-trigger-event 'hover', 'click'
innerZoom Show the zoomed image in the lens instead of popover data-inner-zoom true, false
preload Preloads the large images on page load data-preload true, false
duration Duration for the transition events in milliseconds data-duration Any number value.
Higher value = slower
